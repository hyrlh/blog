这套博客现在已经具备基础可用能力：

- 使用 `Hexo` 生成静态页面
- 使用 `GitHub Pages` 托管站点
- 使用 `GitHub Actions` 自动构建与发布
- 使用本地搜索，不依赖外部搜索服务
- 预留 `giscus` 评论系统接入位

## 当前目录怎么用

常用命令如下：

```bash
npm run server
npm run build
```

## 建议先写的三类文章

1. 自我介绍和博客定位
2. 最近一次有价值的技术排障记录
3. 一篇工具或工作流总结

## 后续建议

- 换一个更适合个人博客的主题
- 补充 About 页面
- 绑定自定义域名
- 开启仓库 Discussions 后补全 giscus 分类 ID

如果你正在看这篇文章，说明博客主流程已经跑通，剩下主要是内容建设和视觉定制。
