类可以理解为把“数据”和“行为”打包在一起的一种自定义类型。

在 C++ 里，类是面向对象编程的核心概念之一。你可以把它先理解成“描述某类对象长什么样、能做什么”的模板。

## 一个最简单的类

```cpp
class Student {
public:
    string name;
    int age;
};
```

这个类定义了一个 `Student` 类型，它有两个成员变量：

- `name`
- `age`

根据这个模板，就可以创建具体对象。

```cpp
Student s;
s.name = "Tom";
s.age = 18;
```

## 类和对象的关系

- 类：像设计图
- 对象：根据设计图创建出来的具体实例

比如“学生”是一个类，而“Tom 同学”是一个对象。

## 成员变量和成员函数

类里不仅可以有数据，还可以有函数。

```cpp
class Student {
public:
    string name;
    int age;

    void introduce() {
        cout << "我是" << name << "，今年" << age << "岁" << endl;
    }
};
```

这里：

- `name`、`age` 是成员变量
- `introduce()` 是成员函数

调用方式：

```cpp
Student s;
s.name = "Tom";
s.age = 18;
s.introduce();
```

## 访问控制

C++ 类里常见的访问权限有：

- `public`
- `private`
- `protected`

初学阶段最常见的是前两个：

- `public`：类外可以直接访问
- `private`：类外不能直接访问

```cpp
class Student {
private:
    int score;

public:
    void setScore(int s) {
        score = s;
    }
};
```

这样做的好处是可以保护内部数据，不让外部随便改。

## 为什么要用类

当程序变复杂后，单独用变量和函数会越来越乱。类的作用是把相关内容组织起来：

- 数据放一起
- 操作这些数据的函数也放一起
- 更容易维护和扩展

这也是为什么后续学习构造函数、封装、继承、多态时，类会成为基础。
